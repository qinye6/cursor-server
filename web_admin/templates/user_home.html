使用说明
  1. 第一步：下载代理补丁
    下载代理补丁
  2. 第二步：解压补丁文件
    将下载的 cursor_代理补丁.rar 文件解压到任意目录
  3. 第三步：根据系统选择安装方式
    Windows 用户：
    • 右键点击 install_win.bat，选择"以管理员身份运行"
    Mac 用户：
    • 打开终端，进入文件所在目录
    • 执行：chmod +x install_mac.sh
    • 执行：sudo ./install_mac.sh
  4. 第四步：配置 Cursor
    1. 选择菜单：文件 → 首选项 → 设置（或按快捷键 Ctrl + ,）
    2. 在"搜索设置"中输入 Proxy
    3. 在 Http: Proxy 下面填入上方的代理地址
    4. 点击保存
    5. 关闭 Cursor 后重新打开即可
